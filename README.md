# StudentManagementSystem
هذا المشروع عبارة عن نظام لإدارة بيانات الطلاب والمدرسين والمقررات الدراسية باستخدام لغة Java وقاعدة بيانات MySQL.
الميزات الرئيسية
• الربط بين قاعدة البيانات والواجهة البرمجية (GUI).
• عرض بيانات الطلاب والدرجات في جدول منظم.
• استخدام SQL Joins لجلب البيانات من جداول متعددة.
التقنيات المستخدمة
• اللغة: Java
• بيئة التطوير: NetBeans
• قاعدة البيانات: MySQL
